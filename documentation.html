Overview

This tool helps match and categorize emails using a validation CSV and a contact list with VANIDs. It supports browser-only functionality with no backend.

How It Works

  1. Upload your Validation CSV (must contain EMAILS and STATUS columns).
  2. Upload your Contacts CSV (must contain PreferredEmail and VANID columns).
  3. Customize filenames for the outputs (optional).
  4. The tool generates three categorized CSVs: VALID, INVALID, and RISKY.

Column Requirements

FileRequired Columns
Validation CSVEMAILS, STATUS
Contacts CSVPreferredEmail, VANID

Status Mappings

The following status strings are automatically mapped to categories:

  • VALID: Valid
  • INVALID: Address Invalid, Domain Invalid, Account Invalid, Mailbox Full, Accept-All, Disposable
  • RISKY: Role Address, Unknown
